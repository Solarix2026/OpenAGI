L4 Capabilities

Beyond conversational AI — a system that acts, remembers, evolves, and anticipates.

Cross-Session Memory

FAISS + SQLite semantic search. Remembers conversations across sessions with importance-weighted retrieval.

Computer Control

A11y Tree + Vision Engine. Controls your desktop like a human — not just API calls. Browser automation included.

Proactive Intelligence

Background monitoring with 24h topic deduplication. Predicts needs and sends relevant nudges.

Self-Evolution

EvolutionEngine detects capability gaps, generates training data, and runs self-tests to improve.

Tool Invention

ToolInventionEngine writes Python tools on-demand and auto-registers them to the tool registry.

Local-First Privacy

Runs entirely on your machine. No data leaves your network. Self-hosted = full control.

Semantic Engine (Groq 8B)

Fast intent classification and routing. Decides if input needs tools, clarification, or direct response.

Tool Executor

Dynamic registry with 35+ tools. Executes shell commands, browser automation, code generation, and more.

NVIDIA NIM 49B

Llama 3.3 Nemotron for all prose generation. Rich contextual responses with memory integration.

Multi-Modal

Porcupine wake word + EdgeTTS voice output. Screen capture + vision analysis for computer control.

Quick Start

Get OpenAGI running in minutes

1

Clone the repository

git clone https://github.com/Solarix2026/OpenAGI.git
cd OpenAGI
2

Install dependencies

pip install -r requirements.txt
3

Configure API keys in .env

GROQ_API_KEY=gsk_xxx
NVIDIA_API_KEY=nvapi_xxx
4

Run OpenAGI

python kernel.py web # Web IDE mode
python kernel.py voice # Jarvis voice mode
python kernel.py # CLI mode

Getting Started

OpenAGI is an L4 autonomous intelligence system designed to be your personal AI assistant with memory, computer control capabilities, and the ability to evolve from interactions.

Project Structure

OpenAGI/ ├── core/ # Kernel, memory, tools, gateway ├── autonomy/ # Proactive, will, habit engines ├── control/ # Computer control, vision, browser ├── agentic/ # Recipes, subagents, DAG workflows ├── evolution/ # Self-improvement, metacognition ├── interfaces/ # Web, voice, Telegram, Jarvis ├── generation/ # SaaS builder, video deck ├── routing/ # Multi-agent router └── safety/ # Guard protocols, MCP adapter

Key Concepts

  • → Semantic Engine: Routes user input to appropriate tools or generates direct responses using Groq 8B.
  • → Memory Core: FAISS vector search + SQLite for episodic storage with importance weighting.
  • → Tool Executor: Dynamic registry where tools can be invented at runtime by the system.
  • → Proactive Engine: Background process that monitors for relevant events and surfaces them.
